# Репозиторий с решениями тестовых заданий для поступления на курс Napoleon IT iOS 2020


### Задание 1
В строке необходимо заменить подстроку, находящуюся между первой открывающей квадратной скобкой и соответствующей ей закрывающей. Исходная строка и новая подстрока, которую необходимо вставить вместо выражения в квадратных скобках, вводятся с клавиатуры. Вы можете использовать любой язык программирования. В ответ прикрепите ссылку на открытый репозиторий с итоговым кодом вашего решения.

Пример:
Входные данные: строка "[Programming [langu[age]]] is the most popular programming language [today]", подстрока: "Javascript".
Выходные данные: "Javascript is the most popular programming language [today]".


### Задание 2
На вход подается число. Необходимо проверить, является ли это число палиндромом (то есть одинаково читается как слева направо, так и справа налево). Нельзя использовать строки и встроенные функции языка программирования. Вы можете использовать любой язык программирования. В ответ прикрепите ссылку на открытый репозиторий с итоговым кодом вашего решения.


### Задание 3
Отсортировать массив чисел таким образом, чтобы сначала шли все четные числа по возрастанию, затем все нечетные по убыванию. Вы можете использовать любой язык программирования. В ответ прикрепите ссылку на открытый репозиторий с итоговым кодом вашего решения.

Пример:
Входные данные: [3, 4, 1, 2, 16, 27, 13].
Выходные данные: [2, 4, 16, 27, 13, 3, 1].


### Задание 4
В репозитории https://github.com/Inxel/NapoleonIT2020 находится стартовый проект. На языке программирования Swift необходимо реализовать методы класса Library. В ответ прикрепите ссылку на открытый репозиторий с итоговым кодом вашего решения.
